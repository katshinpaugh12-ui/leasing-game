Sutton Place Deal Or No Deal - Vercel Upload Folder

Upload this whole folder to GitHub, then import the repo into Vercel.

Included:
- index.html
- src
- public
- package.json
- vite.config.js

Do not upload node_modules. Vercel installs dependencies automatically from package.json.

Vercel settings:
- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist

Saving note:
On Vercel, prizes and winners save in each user's browser. They will not sync across different computers unless a database/backend is added later.
